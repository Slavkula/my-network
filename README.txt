Проект сети на роутерах Cisco. 
Использую инструмент автоматизации настройки Ansible.
Любое изменение буду документировать и проставлять версию сети.

Директории.
PNG - скриншоты топологии, если надо будет что-то визуально отобразить.
Playbooks - использованные плейбуки целиком.
Versions - текстовое описание изменений сети.
out_routers - конфиги внешних роутеров.

Сеть управления: 192.168.1.0/24
Loopback адреса: 172.16.255.0/32
